# 数据库笔记

## mysql

* 事务的并发问题
  * 脏读，读取了回滚事务修改的数据
  * 不可重复读，在另一事务提交前后读取了同一个数据
  * 幻读，多次查询查询记录条数不一致（重点在于insert，不可重复读重点在于update和delete）

* 事务的四种隔离级别
  * 读未提交read-uncommitted
  * 读已提交read-committed
  * 可重复读repeatable-read
  * 串行化serializable
